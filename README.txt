네오스터디 홈페이지 v1

1. 압축을 풉니다.
2. index.html 파일을 더블클릭해서 열어봅니다.
3. 마음에 안 드는 문구/색/배치를 말해주면 수정합니다.

포함 파일:
- index.html
- style.css
- script.js

현재 연결:
- 교재 구매: Google Forms
- 문의: 카카오톡 채널
- 카페: 네이버 카페
- 무료강의: 유튜브


v2 수정사항:
- 실제 네오스터디 가로형 로고를 PC 헤더에 반영했습니다.
- 세로형 로고를 모바일 헤더용으로 반영했습니다.
- 심볼 로고를 favicon 및 푸터에 반영했습니다.


v3 수정사항:
- 저자소개 섹션 추가
- 전남대학교 대학원 기계공학전공 석사 졸업 문구 반영
- 하단 정보에 카카오톡 1:1 문의, 교재 구매 서식, 카페, 유튜브 링크 반영
- 전화번호 없는 문의 구조로 정리


v4 수정사항:
- 사업자 정보 반영
- 전화번호 없는 카카오톡 1:1 문의 구조 유지
- 개인정보처리방침/이용약관 페이지 기본 틀 추가


v5 수정사항:
- 하단 사업자 정보 간소화
- 문의/교재구매/카페/유튜브 반복 노출 제거
- 개인정보처리방침 및 이용약관 링크 유지


v6 수정사항:
- 상단 4개 바로가기 카드 텍스트 가운데 정렬
- 저자소개 문구의 마침표 제거
- 저자소개 안 '저자직강 무료강의 보기' 버튼을 더 눈에 띄게 강조 디자인


v7 수정사항:
- 히어로 우측 이미지를 산업안전기사 필기/실기 교재 표지로 교체
- 교재 구매 바로가기 버튼과 카드 강조
- 저자소개 무료강의 버튼을 더 세련된 블루/화이트 스타일로 변경


v8 수정사항:
- 메인 책 이미지를 산업안전기사 필기/실기 '앞표지 2권' 구성으로 변경
- 책 아래 문구 가운데 정렬 적용


v10 수정사항:
- 메인 페이지에서 저자소개 섹션 제거
- 저자소개를 author.html 별도 메뉴 페이지로 분리
- 상단 카테고리/CTA 영역 세련되게 리디자인
- 4개 바로가기 카드에 아이콘/정렬/스타일 강화
- 교재 섹션 상단 소개 영역을 카드형 디자인으로 개선


v11 수정사항:
- author.html 저자소개에 보유 자격증 전체 목록 추가
- 자격증 목록을 깔끔한 그리드 카드 형태로 정리


v12 수정사항:
- 메인 책 이미지 아래 문구를 '안전보건자격증 필기 · 실기 교재'로 수정
- CTA(교재 구매가 필요하신가요?) 섹션을 더 세련된 스타일로 리디자인
- CTA 문구를 더 간결하고 자연스럽게 수정


v13 수정사항:
- 메인 책 이미지 아래 제목/설명 문구 제거
- 책 표지만 보이도록 단순화


v14 최종 수정사항:
- 메인 페이지 오른쪽 책 비주얼: 책 표지만 표시
- 출간 교재 섹션으로 정리하고 카드 하단 구매 버튼 제거
- 하단 CTA 배너 높이 축소 및 문구 간결화
- 저자소개 페이지 설명에서 '핵심 이론' 제거
- 저자소개 페이지에 보유 자격증 전체 목록 추가


v15 수정사항:
- 출간 교재 카드 여백/높이 정리
- 카드 정렬은 왼쪽 유지
- 출간전 표시를 작은 배지 형태로 정리
- 숨김 구매 버튼 HTML 제거


v16 수정사항:
- '교재 구매 바로가기' 카드에서 MAIN 배지 제거
- 하단 '바로가기 →' 문구 제거
- 다른 카드와 시각 중심이 맞도록 정렬 보정


v17 수정사항:
- 개인정보처리방침 / 이용약관 페이지 하단의 '본 페이지는 홈페이지 공개 전 최종 확인이 필요한 기본 문안입니다.' 문구 제거
